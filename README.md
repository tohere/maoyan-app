## 本地运行查看效果
- 克隆到本地
```bash
git clone https://github.com/tomorrow-here/maoyan-app.git
```
- 进入maoyan-app文件夹运行cmd安装依赖
```bash
yarn
or
npm i
```
- 依赖安装完成后，运行
```bash
yarn dev
or
npm run dev
```
